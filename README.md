# Hitos realizados

## Armas
He escogido las siguientes armas a la hora del desarrollo:
* Bumeran
* FrostNova
* Escudo orbital (Bible)

## Progresión
* Comandos de debug
* Intenté implementar los key bindings para subir automáticamente de nivel las armas (parcialmente hecho).
* Las armas contemplan efectos mejorados al subir de nivel (más proyectiles disparados, más orbes, más daño y CD reducido).
* El jugador sube de nivel y aumenta su vida máxima.
* La experiencia es progresiva también.

## UX

He implementado la mayor cantidad de hitos posibles: 

* Camera shake.
* Números popup con el daño realizado.
* Menú principal.
* Menú de pausa.
* Barra de vida funcional.
